# 针刺 - 快速开始

## 环境准备（只需一次）

### 1. 安装 Android Studio
```bash
brew install --cask android-studio
```

### 2. 安装 JDK 17
```bash
brew install openjdk@17
```

### 3. 配置环境变量
```bash
echo 'export ANDROID_HOME=$HOME/Library/Android/sdk' >> ~/.zshrc
echo 'export PATH=$PATH:$ANDROID_HOME/tools:$ANDROID_HOME/platform-tools' >> ~/.zshrc
source ~/.zshrc
```

---

## 构建 APK

### 方法一：使用脚本（推荐）

```bash
cd /Users/king/Desktop/针刺-app

# 构建 Debug APK（开发测试用）
./build-apk.sh debug

# 构建 Release APK（正式发布用）
./build-apk.sh release
```

### 方法二：使用 Android Studio

1. 打开 Android Studio
2. 选择 "Open" → 找到 `针刺-app` 文件夹
3. 等待 Gradle 同步完成（首次约 10-30 分钟）
4. 点击菜单栏 **Build → Build Bundle(s) / APK(s) → Build APK(s)**

---

## 安装到手机

### 1. 开启 USB 调试
- 手机设置 → 关于手机 → 连续点击"版本号"7次
- 返回 → 开发者选项 → 开启"USB 调试"

### 2. 连接手机
```bash
# 检查设备是否连接
adb devices

# 安装 APK
adb install app/build/outputs/apk/debug/app-debug.apk
```

### 3. 或者手动安装
- 将 APK 发送到手机
- 点击安装（需要允许"未知来源"）

## 📁 APK 位置

构建完成后，APK 文件位于：

```
针刺-app/
└── app/
    └── build/
        └── outputs/
            └── apk/
                ├── debug/
                │   └── app-debug.apk          ← 测试版
                └── release/
                    └── app-release-unsigned.apk  ← 发布版（需签名）
```

## 🐛 常见问题

### 1. Gradle 同步失败
```bash
# 清理缓存
./gradlew clean

# 重新同步
./gradlew build --refresh-dependencies
```

### 2. 构建失败
```bash
# 检查环境
java -version
echo $ANDROID_HOME

# 确保使用正确的 JDK
export JAVA_HOME=$(/usr/libexec/java_home -v 17)
```

### 3. 安装失败
```bash
# 卸载旧版本
adb uninstall com.zhenci.app

# 重新安装
adb install app/build/outputs/apk/debug/app-debug.apk
```
